Portfolio stuff I made

Willy GitHub npm

Willy is an assertion library designed to be simple and readable. Chai and Should didn't feel as natural as they aimed to be and ended up distracting from the code. While studying Mocha, I thought it would be fun to write my own.

Texas Ranger GitHub npm

This is an npm module for walking directories in search of files with a certain extension. It works asynchronously and recursively and uses the module pattern to expose global module settings.
